Use CSS to set the background color of the document to yellow.

    <html>
    <head>
      <style>
        body {
          background-color: yellow;
        }
      </style>
    </head>
    <body>
    
    <h1>This is a heading</h1>
    
    <p>This is a paragraph.</p>
    
    </body>
    </html>
