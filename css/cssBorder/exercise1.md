Use the border shorthand property to set a "4px", "dotted", "red" border for the <p> elements.

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        p {
          border: 4px dotted red;
        }
      </style>
    </head>
    <body>
      <h1>This is a heading</h1>
      <p>This is a paragraph</p>
      <p>This is a paragraph</p>
    </body>
    </html>
