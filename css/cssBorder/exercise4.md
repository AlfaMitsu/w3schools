Use the correct border property to add ROUNDED borders to the <p> elements.

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        p {
          border: 2px solid red;
          border-radius: 5px;
        }
      </style>
    </head>
    <body>
      <h1>This is a heading</h1>
      <p>This is a paragraph</p>
      <p>This is a paragraph</p>
    </body>
    </html>
