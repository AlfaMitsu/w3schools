Use the correct border property to set the border color to "red".

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        p {
          border-style: dotted;
          border-width: 4px;
          border-color: red;
        }
      </style>
    </head>
    <body>
      <h1>This is a heading</h1>
      <p>This is a paragraph</p>
      <p>This is a paragraph</p>
    </body>
    </html>
