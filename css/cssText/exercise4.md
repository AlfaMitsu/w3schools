Style text in <h1> to uppercase letters, and text in <p> to capitalized letters.

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        h1 {
          text-transform: uppercase;
        }
        p {
          text-transform: capitalize;
        }
      </style>
    </head>
    <body>
      <h1>This is a heading</h1>
      <p>this is a paragraph</p>
    </body>
    </html>
