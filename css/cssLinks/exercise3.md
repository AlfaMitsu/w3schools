Set the style for links to pages you have visited to red.

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        a:visited {
          color: red;
        }
      </style>
    </head>
    <body>
      <h1>This is a heading</h1>
      <p>This is a paragraph</p>
      <a href="http://w3schools.com">This is a link</a>
    </body>
    </html>
