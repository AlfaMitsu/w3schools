Set the text color to red, for all <p> and <h1> elements. Group the selectors to minimize code.

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        h1, p {
          color: red;
        }
      </style>
    </head>
    <body>
      <h1>This is a heading</h1>
      <h2>This is a smaller heading</h2>
      <p>This is a paragraph</p>
      <p>This is a paragraph</p>
    </body>
    </html>
