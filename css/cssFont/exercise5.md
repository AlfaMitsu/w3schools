Use the correct font- property to style the <p> elements as "small-caps".

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        p {
          font-variant: small-caps;
        }
      </style>
    </head>
    <body>
      <h1>This is a heading</h1>
      <p>This is a paragraph</p>
      <p>This is a paragraph</p>  
    </body>
    </html>
