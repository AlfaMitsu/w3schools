Set the list style to I, II, III, instead of 1, 2, 3 for ordered lists.

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        ol {
          list-style-type: upper-roman;
        }
      </style>
    </head>
    <body>
      <ol>
        <li>Coffee</li>
        <li>Tea</li>
        <li>Coca Cola</li>
      </ol>
    </body>
    </html>
