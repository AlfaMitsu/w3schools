Set the background-color to red, when you mouse over elements with the class "master".

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        .master:hover {
          background-color: red;
        }
      </style>
    </head>
    <body>
      <h1 class="master">This is a header.</h1>
      <p class="master">This is a paragraph.</p>
      <p>This is a paragraph.</p>
    </body>
    </html>
